又到了回顾2012的时候啦，这一次依旧是有些惭愧滴，又有半数没有完成啊 ... 不过今年倒不算是太偷懒， 中途变更了许多想法，倒是做了颇多额外的东西。

一度在摇摆是认真看看C语言呢，还是弄弄新东西，最后还是选择把C看好先，不过这个“好”离“用”的距离还差了100个Java。日语学习也摆在了最后，而定了更严格得英语学习计划，毕竟感觉英语价值更高些，而且离专业级得英语专家差距还不是一星半点。后来又有感于电影看得太多了点，定下个期望读100本书的计划，完成的也不错，抛开了漫画，去年看了98本，基本完成了目标 ^_^ 后来因为看原版的《冰与火》，进度慢了许多，要不真可能完成计划。技术上方面，在VIM和Git上都花了不少的时间，可VIM的熟练度离用来做开发还有距离，而Git离搭建一个公司的内部管理也还有差距 ... 最值得肯定了的是，后面几个月开始坚持跑步，再一次逐渐形成了习惯。感觉逐渐再懒散，习惯安逸的时候，这真是一个可贵的积极信号，I still can do it ~

回想起来总觉得做的事情不够多，很直观的说明动手的东西还是太少，拿出的成品不够。网盘里分享的PPT虽然积累的还是多了起来，可要有更多的代码就好了，这就是今年的主要目标了。

迈过了30的门槛，这个新的起步算不得优异，倒还不是那么难看。之后的压力或者就会逐渐大起来了，最为重要的就是为自己买一个永不过期的“保险”。这几年最大的感受就是，只会一种语言是会死的，只会Java更是危险的。现在的经济算不上好，这种几乎只适用于大公司的编程语言，在快速求变的当下和未来都带来很大的风险。更不是一个自由职业者最好的选择。我要做的只是把现在为止起步的东西都切实得用起来就好。

  • 完成 One Library (Php，Javascript, Html5 和 Android)
  • 每2周用C语言实现一个算法，记录日志
  • 每2周研究一个Java Pattern，记录日志
  • Master Git
  • Master Vim
  • Everyday English
  • 每天1小时跑步
  •  Study《深入理解计算机系统》
  • Read《 A Song of Ice and Fire》Englsih Version
  • Read《1984》English Version
  • 了解 Object-C  开发
  • 了解 Lisp

毫无疑问，计划跑不赢变化。所有我期望做到得就是，减少一些看电影，每周能看2-3部就差不多了。同时也减少些发神经得时间，尽可能把时间都挤出来写代码，做些明年啃一真切来出来把玩得东东。2014见！
